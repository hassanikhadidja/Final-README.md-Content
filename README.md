My E-Commerce Website - MERN Stack

A full-stack e-commerce platform built with the **MERN Stack** (MongoDB, Express.js, React.js, Node.js) as part of my Bootcamp final project.

Project Type: Full-Stack Web Application

> 🚀 Project Overview

This project is a complete e-commerce website where users can browse products, add items to cart, register/login, and complete orders. It includes both **customer-facing** features and a basic **admin dashboard** for managing products and orders.

The goal was to apply everything learned during the Bootcamp in one real-world style application.

> 🛠 Skills Gained During the Bootcamp

This Bootcamp has been a transformative experience. Here are the most important skills I developed:

- **Full-Stack Development**: Building complete applications from database to frontend using the MERN stack.
- **Backend Development**: Creating RESTful APIs with Node.js and Express, handling routes, controllers, middleware, and error handling.
- **Database Management**: Designing schemas with MongoDB and Mongoose, performing CRUD operations, and understanding NoSQL concepts.
- **Frontend Development**: Building dynamic and responsive user interfaces with React.js (functional components, hooks, React Router).
- **Authentication & Security**: Implementing JWT-based authentication, password hashing with bcrypt, and protected routes.
- **State Management & API Integration**: Connecting frontend to backend using Axios, managing global state (Context API or Redux), and handling asynchronous operations.
- **Modern Web Practices**: Using Tailwind CSS / Material UI for responsive design, form validation, and clean code structure.
- **Version Control & Collaboration**: Professional use of Git and GitHub (commits, branches, README documentation).
- **Problem-Solving & Debugging**: Breaking down complex features, troubleshooting bugs, and thinking like a developer.

I also improved my **time management**, **self-learning ability**, and confidence in working on large projects independently.
